# chat_analyser
A Python project that analyzes WhatsApp group chats to discover activity patterns, personalities, emoji usage, friendships, and conversation statistics.
## Constraints

| Day   | Work Done                                        |
| ----- | ------------------------------------------------ |
| Day 1 | Parsed WhatsApp chat file                        |
| Day 2 | Cleaned messages and extracted users             |
| Day 3 | Built activity analysis                          |
| Day 4 | Added emoji and media analysis                   |
| Day 5 | Added friendship and personality metrics         |
| Day 6 | Designed final report formatting                 |
| Day 7 | Testing, debugging, documentation, GitHub upload |
